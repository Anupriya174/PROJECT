DSA Problem Tracker
A simple web application to track Data Structures and Algorithms (DSA) problems you solve.

Project Description
This project allows users to add DSA problems along with their difficulty level (Easy, Medium, Hard).
Users can also mark problems as Solved after completing them. The application dynamically updates the list of problems.

Features
Add new DSA problems.
Select problem difficulty.
Mark problems as solved.
Simple and interactive user interface.

Technologies Used
HTML
CSS
JavaScript

Project Structure
index.html -main webpage 
style.css -styling for the webpage
script.js -javascript logic for adding and tracking problems
